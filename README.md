# mirorea_excavator_cad

3D printable CAD parts for the Mirorea Excavator.

MiRoREA Excavatorの3Dプリント部品のCADデータ

## Documentation

Instructions for 3D printing and assembly are provided in the following document:
https://open-mirorea.github.io/mirorea_excavator/

## Contents

- `cad/` – FreeCAD source files (.FCStd)
- `step/` – STEP export files for CAD interoperability
- `stl/` – STL files for 3D printing

## Notes

### General Notes

The canonical source of this design is this repository:
https://github.com/Open-MiRoREA/mirorea_excavator_cad

If you redistribute this design or derivatives, please provide a link to this repository so users can access the original source files.

These parts are designed for the MiRoREA Excavator developed as part of the Open MiRoREA project.
Compatibility with other systems is not guaranteed.

### Technical Notes

The STEP and STL files are generated from the FreeCAD source files.
If you modify the design, please edit the files in the `cad` directory.

Due to licensing restrictions of the original CAD data, all models in this repository have been redesigned for public release.
Although the dimensions are intended to be compatible with the previously used parts, some minor geometric changes were introduced due to CAD implementation constraints.
The geometry of the main features has been checked against the original STL models.
However, as of March 2026, the parts in this repository have not yet been physically verified through 3D printing and assembly.
